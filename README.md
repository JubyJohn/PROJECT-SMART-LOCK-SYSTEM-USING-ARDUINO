# PROJECT SMART LOCK SYSTEM


## AIM

To create a smart lock  system using LCD and keypad interfaced with arduino.


## COMPONENTS

1) Arduino UNO R3
2) LCD 16*2 Module
3) Keypad 4*4
4) Jumper Wire
5) USB Cable


## CONNECTION

### Matrix 4*4 keypad Module
 
<br> R1 ---->  10			C1 ---->  6
<br> R2 ---->  11			C2 ---->  7
<br> R3 ---->  12			C3 ---->  8
<br> R3 ---->  13			C4 ---->  9	

### LCD 16*2  Module
 

<br> RS ---->  A0			Power supply  ---->  5V
<br> EN ---->  A1			GND   ---->  GND
<br> D4 ---->  A2			D6 ---->  A4
<br> D5 ---->  A3			D7 ---->  A5
